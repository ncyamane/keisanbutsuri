# keisanbutsuri

計算物理の実装

## 内容(予定)

概ね以下の内容を予定している.

- 初歩的な数学の復習
- 数値計算入門
- プログラミング言語速習
  - C/C++
  - julia
  - Fortran
- アルゴリズム入門
- 非線形方程式の解法
  - 2分法
  - ニュートン法
  - 多変数ニュートン法
- 数値線形代数
  - 連立一次方程式とガウスの掃き出し法
  - 固有値問題と特異値問題
  - LU分解
  - べき乗法
- 微分方程式の解法
- 数値積分
- データ解析
- 関数の近似・補間
- フーリエ変換
- 並列計算・GPUの利用
